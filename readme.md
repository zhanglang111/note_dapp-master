## 前制条件

安装Ganache、 Truffle、 MetaMask

1. 安装库
npm install 

2. 启动Ganache 

3. 编译部署 
truffle migrate

部署到以太坊网络：使用自己的助记词 和 Infura 访问key 配置 truffle.js， 然后：

truffle migrate --network ropsten

4. 启动
npm run dev 

5. http://localhost:3000 查看效果
